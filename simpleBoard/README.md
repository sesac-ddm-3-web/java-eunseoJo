# 요구사항 정의 
- 회원, 게시글, 댓글 엔티티 
- ID/Password 입력하여 회원가입과 로그인 가능
- 로그인 한 후에만 이용가능 
- 프론트엔드를 포함한 서비스 

## 게시글
- 제목, 내용, 작성자, 조회수, 작성일시
- 게시글 목록에서는 내용을 제외하고 보여준다.
- 게시글 누르면 게시글 내용과 해당 게시글에 달린 댓글 볼 수 있다.
- 게시글에서는 댓글 작성 가능
- 게시글, 댓글 삭제는 본인만 권한 

## 댓글
- 내용, 작성자, 작성일시
